Privacy Policy

Last updated: 2025

Overview

Simple Color Mixer is a Chrome extension that allows users to mix and preview colors directly within a browser popup.

Developer Information

Developer: Rohan R. K.
Contact: rohankadam2596 [at] gmail [dot] com

Data Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data.

Data Processing

All color calculations and previews are performed locally in the browser. No data is sent to external servers.

Permissions

This extension does not request or require any special permissions.

Third-Party Services

Simple Color Mixer does not use any third-party services, analytics, advertising networks, or external APIs.

Offline Use

The extension works fully offline once installed.

Changes

Any updates to this privacy policy will be reflected on this page.
